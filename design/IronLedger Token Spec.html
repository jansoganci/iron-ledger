09 Changelog from v1 What changed, what stayed, what was retired.
NEW — Amber Scale
Added full 50–950 amber scale as the new primary action color. Replaces teal as the hero CTA color. Semantic tokens: --accent-primary, --accent-primary-dark, --accent-primary-bg, --accent-primary-text.
NEW — Emerald Scale
New 50–950 emerald scale for reward moments and success states. Warmer and more saturated than the previous #166534-family. Used for --severity-favorable and confirm/close actions.
NEW — Violet Scale
New 50–950 violet scale exclusively for agentic and processing UI states. Pulse animations, data flow bars, processing indicators. Semantic tokens: --accent-agent family.
REVISED — Neutral Scale
Canvas/surface base unchanged (#FAFAF8 ≈ original). Mid-range tones slightly warmer (+hue 60 vs 0). Border tokens now use --neutral-200 and --neutral-300 for clear hierarchy.
REVISED — Severity / Medium
Medium severity migrated from old #FEF3C7 / #B45309 to amber-family tokens for brand cohesion. High severity background lightened slightly to #FEF0F0 (less aggressive). Favorable now uses emerald instead of old #ECFDF5 / #166534.
DEMOTED — Teal
Teal (#0D9488) retired from primary accent role. Kept as --accent-utility for secondary links, data table header text, inline highlights. Not for buttons or primary UI chrome.
NEW — Motion Token Set
5 duration tokens (80–650ms), 4 easing curves, 3 scale values, 3 accent shadows, and @media (prefers-reduced-motion) safety layer. Zero Framer Motion dependency — pure CSS custom properties.
REVISED — Typography
Source Serif 4 promoted to display/hero-number role (was underused). Inter weight scale tightened: 700 for H1, 600 for H2–H3, 500 for labels/UI. JetBrains Mono now specified for all data cells with font-variant-numeric: tabular-nums.
